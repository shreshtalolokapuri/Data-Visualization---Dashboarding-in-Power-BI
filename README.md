# Sales Dashboard - Power BI Project  
  
**Overview**  
This project presents an interactive Sales Dashboard built using Microsoft Power BI to analyze sales performance, customer trends, shipping insights, and regional distribution.  
The dashboard provides a clear visualization of key business metrics to support data-driven decision making.  
  
**Dashboard Highlights**   
  
**Key Metrics**  
Total Sales Amount: 11.99M  
Average Sales Amount: 1.78K  
Total Orders Count: 6727  
  
**Visualizations Included**  
Sales trend by year  
Sales distribution by order priority  
Sales contribution by salesman  
Shipping mode analysis  
Category-wise sales analysis  
State-wise sales distribution map  
Sales amount by customers  
Average shipping days by state

**Business Insights**  
  
This dashboard helps identify:  
  
- High-performing sales categories  
- Top customers and sales representatives  
- Shipping performance trends  
- Regional sales distribution  
- Order priority contribution to revenue
